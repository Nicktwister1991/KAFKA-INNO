## Инструкция
Установка кафки:
- Установка докера — https://www.docker.com/products/docker-desktop/
- Создайте папку, где у вас будет храниться образ ,например kafka 
- Создаем docker compose - в нашем случае он уже создан и называется docker-compose
- Разворачиваем контейнеры. Заходим в командную стройку, выбираем путь, где лежит docker compose пример команды - cd C:\Users\User\Desktop\KAFKA (в вашем случае путь где он лежит)
- Запуск контейнеров - docker-compose up -d
- Создаем брокеры и топики. Для win нажимаем на файцл - create-topic. Для linux и mac create-topic-linux (в файле под названием команды описано как его запустить)
- Открываем docker-desktop, заходим в Containers. Выбираем ui, переходим по ссылке и попадаем в панель управления кластерами кафки

Установка продюсера и консюмеров:
- В папке kafka файл для установки среды разработки если еще ее нет. При установке ставим галочу Intelejie IDEA,Add "bin",Add "Open folder" и .java
- Делаем все как на видео.
